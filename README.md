# Fiche Profil Python

Mon premier projet Python, une fiche profil interactive en ligne de commande.

## Ce que fait le programme
- Utilisation de variables pour stocker les informations des utilisateurs
- Liste avec fonction enumerate() avec u index+1 pour afficher les langages en commencant par 1 et non zero
- Utilisation d'une boucle pour mettre autant de langages possibles
- Emploie de conditions if, elif, else pour déterminer laa majorité ou non de l'utilisateur et le niveau de développeur
  selon le nombre de langages
- Affiche une fiche profil complète

## Langages utilisés
- Python

## Comment lancer
```bash
python jour1.py
```
